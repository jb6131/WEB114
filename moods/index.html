Project 7 -- DOM and JS

JavaScript Questions

1. Why is textContent safer than innerHTML?

Answer: textContent is safer than innerHTML because textContent only takes in plain text and will not parse through or run any code, even if someone tries using a script tag.

2. How are HTML elements connected to JavaScript variables using DOM selection?

Answer: HTML elements are connected to JavaScript variables using DOM selection such as getElementById or querySelector. They search the page for the desired element and return it.

3. What would happen to the text box with id="text" if I added this line to the code above it:

text.textContent = "Hello!"

Answer: If you were to add that line, the content of the text box would be replaced with "Hello!".
